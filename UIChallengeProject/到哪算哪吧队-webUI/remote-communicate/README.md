# 到哪算哪吧队-webUI

## 项目简介

这是基于vue进行开发的web端视频界面，项目名称是`Remote Communicate`简称RC，项目logo也是基于RC进行的变形。

## 项目预览

可以直接打开根目录下面`dist`文件夹中的`index.html`。

### 登录信息：
```
App ID： myid
Channel: mychannel
Token: mytoken
```

## 运行项目
```
npm install
```
### 有可能需要手动安装sass依赖包
```
npm i node-sass sass-loader --save
```

### 启动项目
```
npm run serve
```
